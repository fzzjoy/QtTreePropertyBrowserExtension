Property Browser Framework Examples
===================================

This package contains the source code for the examples described in the
"The Property Browser Framework" article in Qt Quarterly 18.

Each of these examples can be configured and built using qmake and make.
Before building an example, you need to replace the placeholder in its .pro
file with the path to the Property Browser solution on your system.
